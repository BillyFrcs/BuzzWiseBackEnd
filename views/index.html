Users

Sign Up

  • URL
    • /api/users/signUp
  • Method
    • POST
  • Request Body
    • name as text
    • email as text, must be unique
    • password as text, must be at least 6 characters
  • Response 
            
    {
        "message": "User Account Signed Up Successfully"
    }

  • URL

  • Request Body
